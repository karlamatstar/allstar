## 1. 문서 사용 방법

이 문서는 프로젝트 전체 진행 상황을 한눈에 확인하는 기준 체크리스트다. 기능을 구현하거나 검증·문서화·Git 반영 상태가 바뀌면 관련 기준 문서와 함께 이 문서도 갱신한다.

- `[x]`: 현재 코드 또는 문서에 반영되어 완료됨
- `[ ]`: 아직 해야 하거나 다시 확인해야 함
- `부분 완료`: 일부 구현됐지만 완료 기준 전체를 충족하지 못함
- `실제 검증 필요`: 목·단위 테스트는 통과했지만 실제 외부 API 또는 실제 서비스 환경 확인이 남음
- `문서만 완료`: 요구사항은 확정했지만 실제 코드는 아직 변경하지 않음

완료 표시는 다음을 구분한다.

1. 구현 완료
2. 비API 자동 검증 완료
3. 실제 API·서비스 검증 완료
4. 관련 문서 갱신 완료
5. Git 커밋·푸시 완료
